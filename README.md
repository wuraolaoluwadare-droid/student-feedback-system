# Online Student Feedback Evaluation System

## Project Overview

The Online Student Feedback Evaluation System is a web-based application developed using Django and MySQL. It enables students to evaluate lecturers, submit anonymous complaints, and view their feedback history, while administrators manage academic records and analyze feedback.

## Features

- Student Login
- Student Dashboard
- Lecturer Evaluation
- Anonymous Complaint Submission
- Feedback History
- Student Profile
- Logout
- Administrator Management
- MySQL Database Integration

## Technologies Used

- Python 3.14
- Django 6
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

1. Clone or download the project.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Configure the MySQL database.

4. Run migrations:

```bash
python manage.py migrate
```

5. Start the server:

```bash
python manage.py runserver
```
